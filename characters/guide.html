A Small Sandbox of Quote Annotation

Main Annotation Tool Page

Operational

  • To create a new character, click the "Add Character" button. YOur new character will be available from now on!
  • To create a new quote/mention, highlight a span of text and select the appropriate options in the modal that pops up. You can exit this modal at any time.
  • To edit an existing quote/mention, click on it. An edit modal will popup where you can change whatever needs changing.
  • To create a new connection between quotes/mentions, ctrl(or commmand)+click the first span(quote or mention), then ctrl+click the one that it should be connected to. Blue lines will appear between them. Connections cannot be editted once they've been created.
  • To delete anything (quote, mention, connection, etc.), simply alt+click the span or connection. If you alt+click a span that has a connection, that connection will also be deleted
  • To begin annotating, the recommended method is to load a text or xml file using the file-chooser.
  • To save, enter the filename in the save box and click save, this will save to your machine. You can continue editting this file later by loading it using the load function (everything, including the characters that you are using will be loaded).

Annotation guidelines

  • Each quote should be annotated with the character that is that quote's speaker.
  • Each quote should be linked to a mention that is the most obvious indication of that quote's speaker.
    • Quotes can be linked to mentions inside other quotes (e.g. "What have you found? Are they Gooseberries?" is linked to "Mathilda" in the preceeding quote).
    • Multiple quotes may be linked to the same mention.
  • Mentions should also be annotated with the character that they refer to.
    • If a character's name is meaningfully associated with an article (e.g. "the Bear"), include that article in the mention.
