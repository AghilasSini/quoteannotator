Annotation Guidelines

Instructions

    • Option 1: copy and paste text into the text area, then click the "Annotate!" button (green).
    • Option 2: load a text file. Annotation mode will be automatically triggered thereafter.
  2. Annotate away! To make a quote, simply highlight it.
  3. After selecting a quote, a box will appear to select the quote's speaker. You can either click or use the number keys (and then the return key to submit) to select the proper speaker.
  4. If you have made a mistake, hold down the "alt" key and click the quote to remove it.
  5. Done? Want to take a break? Enter a filename to save to like "quotes4dayzzz". You will be prompted to download "quotes4dayzzz.xml". If you want to continue annotating this file at a later time, simply load it using the load function.

Finished with your file? Questions? Bugs? Want more functionality? The answer is simple: email me!

Known Issues

  • Funkiness upon reload. To fix this, do a hard reload (shift+command/apple+r). You will lose the text in the text box and cannot save it!
